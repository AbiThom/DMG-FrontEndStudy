Git Cheat Sheet
Git Key

Git commands: pwd = shows where you are ls = list of where you can go cd + "file name"/= navigates you to said file if it exists cd .. = takes you back to parent folder code . = opens vs code mkdir + "file name" = create directory touch index.html= creates a html file git add . = adds changes to working area to staging area git push= uploads local repository content to remote repository git commit = saves your progress mv ./"file" ./"file"/ = moves folder or file somewhere
